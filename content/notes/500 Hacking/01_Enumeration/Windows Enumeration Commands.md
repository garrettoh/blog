

Gather System information `systeminfo`

Check installed updates `wmic qfe get Caption,Description`

Check Services `net start`'

Installed apps `wmic product get name,version,vendor`

See the user and privileges `whoami /priv`
User groups `whoami /groups`

Enumerate users `net user`

Enumerate groups `net group` or `net localgroup`

Enumerate users in a group `net localgroup administrators`

View active connections with PID and Name `netstat -abno`
